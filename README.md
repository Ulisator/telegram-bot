ChilePoolBot (BETA)
===================

Bot para Telegram que entrega informacion relevante sobre la red de varias Cryptomonedas Chilenas.

### Cryptos soportadas
- Prosus
- Luka
- Ento

### Herramientas de desarrollo
* Node v7
* telegram-node-bot
* request
* numeral

### Como correrlo

Instalar las dependencias
```sh
        npm install
```

Luego ejecuta
```sh
        npm run start
```
